# Math

## Description

This repository contains implementations of mathematical concepts and algorithms used in machine learning and artificial intelligence.

The projects focus on building mathematical operations from first principles using Python and NumPy while developing a deeper understanding of the underlying theory.

## Projects

* Linear Algebra
* Calculus
* Plotting
* Probability
* Convolutions and Pooling
* Classification
* Optimization
* Error Analysis
* Multivariate Probability
* Hidden Markov Models
* Advanced Linear Algebra

## Technologies

* Python 3
* NumPy

## Requirements

* Ubuntu 16.04 LTS
* Python 3.5
* NumPy 1.15
* Pycodestyle 2.5
* All modules, classes, and functions are documented.
* Only the allowed Python modules are imported for each project.

## Learning Objectives

The projects in this repository cover concepts including:

* Linear algebra operations
* Probability theory
* Calculus fundamentals
* Matrix manipulation
* Image processing
* Convolutional operations
* Pooling techniques
* Mathematical foundations of machine learning
